# **App Name**: Nature Kabana

## Core Features:

- SEO-Optimized Pages: Generate and serve SEO-friendly meta tags and content for all pages.
- Dynamic Room Showcase: Showcase room and cabana types with demo images, descriptions, and direct booking links.
- Interactive Gallery: Display a gallery with nature/hotel images using a responsive grid and lightbox viewer.
- Booking.com and other services Integration: Integrate buttons and links to Booking.com, Agoda, and Airbnb for external booking.
- Inquiry tool via WhatsApp: Enable direct customer inquiries and support through a floating WhatsApp button.
- Internal Booking Tool: Enable internal booking via form, that submits the request to Firestore database.
- Dynamic Content Generation tool: Use AI to generate content variations and suggestions for the 'About Us', 'Nature Garden', and 'Things To Do' sections, enhancing SEO and user engagement by the tool.

## Style Guidelines:

- Primary color: Forest Green (#228B22) to evoke nature and tranquility.
- Background color: Light Beige (#F5F5DC), providing a calm and neutral base.
- Accent color: Earth Brown (#A0522D) for a natural and grounding touch.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable, and neutral design.
- Use natural, organic icons that represent the hotel's themes, like leaves, water, or sun.
- Employ a clean, minimal layout that emphasizes spaciousness and nature-focused imagery.
- Utilize smooth, subtle animations using Framer Motion for transitions and interactions to enhance user experience.
